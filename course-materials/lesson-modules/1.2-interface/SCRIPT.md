Let's talk about **execution modes** - how the agent approaches tasks

Antigravity has two main modes:

**Fast Mode** (what you're using now)
- Agent responds directly to your requests
- Quick back-and-forth conversation
- Best for simple tasks, questions, quick edits

**Planning Mode**
- Agent creates a detailed plan before acting
- Shows you what it will do before doing it
- Best for complex, multi-step work
- We'll cover this in depth in Module 1.5!

For now, stick with **Fast mode** - it's perfect for learning the basics
