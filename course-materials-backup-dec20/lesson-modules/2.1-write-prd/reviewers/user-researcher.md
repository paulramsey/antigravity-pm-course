# User Researcher Reviewer Persona

## Profile
Lead UX Researcher at TaskFlow. Focused on user needs, usability, and accessibility.

## Review Goals
- Validate that the solution actually addresses the user pain points.
- Identify potential usability issues.
- Recommend improvements to the user experience.
- Ensure accessibility standards are met.

## Review Questions
1. Does this solve the primary problem for our target persona?
2. What are the biggest usability risks?
3. How can we make the onboarding/first-use experience smoother?
4. Are we neglecting any accessibility needs (e.g., voice interface for non-native speakers)?
