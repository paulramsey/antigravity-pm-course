# Module 0.2: Installation & Setup
## Reference Guide

- **Time to Complete:** 10 minutes
- **Prerequisites:** Mac (10.15+), Windows (10/11), or Linux

## Overview

Install Google Antigravity on your computer and get ready to start the course.

**Key takeaway:** Download from antigravity.google/download, install, and you're ready to go. No account or subscription required - Antigravity is free during public preview.

## Installation

### Mac

1. **Download Antigravity**
   - Go to [antigravity.google/download](https://antigravity.google/download)
   - Click "Download for Mac"
   - Wait for the installer to download (~150MB)

2. **Install**
   - Open the downloaded `.dmg` file
   - Drag Antigravity icon to Applications folder
   - Eject the installer

3. **First launch**
   - Open Applications folder
   - Double-click Antigravity
   - If you see a security warning → See troubleshooting below

### Windows

1. **Download Antigravity**
   - Go to [antigravity.google/download](https://antigravity.google/download)
   - Click "Download for Windows"
   - Wait for the installer to download (~150MB)

2. **Install**
   - Run the downloaded installer file
   - Follow the installation wizard
   - Choose "Install for current user" (recommended)
   - Accept default install location

3. **First launch**
   - Open from Start menu or desktop shortcut
   - Windows Defender may scan on first run (normal)

### Linux

1. **Download Antigravity**
   - Go to [antigravity.google/download](https://antigravity.google/download)
   - Download the appropriate package for your distribution

2. **Install**
   - Follow the installation instructions for your Linux distribution
   - Make the application executable if needed

3. **First launch**
   - Launch from your applications menu or terminal

## First Launch Setup

When Antigravity opens for the first time:

1. **Choose your theme**
   - Select Light or Dark (you can change this later)
   - Click "Continue"

2. **Sign in (optional)**
   - You can sign in with a Google account for cloud features
   - Or continue without signing in for local-only use

3. **Welcome screen**
   - You'll see a brief tutorial overlay
   - Click through or press "Skip" to dismiss

## Troubleshooting

### Mac: Security warning when opening

**Fix:**
1. Open **System Preferences** → **Security & Privacy**
2. Click the **Privacy** tab
3. Look for message about Antigravity being blocked
4. Click **"Open Anyway"**
5. Relaunch Antigravity from Applications

### Slow Download or Timeout

**Fix:**
- Download may take 5-10 minutes on slower connections
- Try a different browser (Chrome/Firefox/Safari)
- Download directly from [antigravity.google/download](https://antigravity.google/download)
- Check your internet connection

### Installation Failed or Won't Complete

**Mac fix:**
- Make sure you have ~500MB free disk space
- Check Applications folder permissions
- Try dragging to Applications again

**Windows fix:**
- Run installer as Administrator (right-click → Run as Administrator)
- Disable antivirus temporarily during install
- Make sure Windows is updated

**Linux fix:**
- Ensure you have the correct package for your distribution
- Check file permissions
- Consult Antigravity documentation for your specific distro
