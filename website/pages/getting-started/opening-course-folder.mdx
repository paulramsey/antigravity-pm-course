## Unzip Course Files

**Mac:**
1. Open Downloads folder
2. Double-click the ZIP file
3. A new folder appears: `complete-course`
4. Drag this folder to your Documents folder (or preferred location)

**Windows:**
1. Open Downloads folder
2. Right-click the ZIP file
3. Select **Extract All**
4. Choose destination (Documents folder recommended)
5. Click **Extract**

> âš ï¸ **Important:** Remember where you put this folder. You'll open it in Antigravity next.

## Open Folder in Antigravity

You have three ways to open the course folder in Antigravity:

### Method 1: File Menu

1. Launch Antigravity
2. Click **File â†’ Open Folder** (Mac) or **File â†’ Open Folder** (Windows)
3. Navigate to where you unzipped the files (e.g., Documents)
4. Select the `complete-course` folder
5. Click **Open**

### Method 2: Drag and Drop

1. Launch Antigravity
2. Open Finder (Mac) or File Explorer (Windows)
3. Navigate to the `complete-course` folder
4. Drag the entire folder onto Antigravity's window
5. Antigravity opens the folder as a project

### Method 3: Welcome Screen

1. Launch Antigravity
2. On the welcome screen, click **Open Folder**
3. Navigate to `complete-course`
4. Click **Open**

## Trust the Folder

When you open the folder, Antigravity may ask:

> **"Do you trust the authors of files in this folder?"**

Click **Yes, I trust the authors**.

**Why this appears:** Antigravity protects you from potentially malicious code. Since you downloaded these course files from the official website, they're safe to trust.

## ğŸ› Troubleshooting

### "I don't see the File Explorer"
**Fix:** Press `Cmd+B` (Mac) or `Ctrl+B` (Windows) to toggle the sidebar. Or click View â†’ Sidebar.

### "Slash commands don't appear when I type /"
**Fix:** Make sure you opened the folder (not individual files). Close Antigravity, then use File â†’ Open Folder and select the entire `complete-course` folder.

### "The folder structure looks different"
**Fix:** You may have opened a subfolder by mistake. Close Antigravity and open the top-level folder (the one containing `.agent`, `company-context`, and `lesson-modules`).

### "Antigravity asks me to trust the folder every time"
**Fix:** Make sure you clicked "Yes, I trust the authors" and checked "Trust all files in parent folder" (if that option appears).
